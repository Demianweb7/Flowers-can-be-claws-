# Flowers can be claws🐾
A simple healing and relaxing small web game. Dedicated to my partner.
A pixel-art browser game where a cat roams a garden and pounces on flowers.

## How to Play

Open `Flowers can be claws🐾.html` in any modern browser.

| Key | Action |
|-----|--------|
| W / A / S / D | Move the cat |
| Space | Pounce on nearby flowers |
| Reset Garden button | Regrow all flowers |

## Features

- **8 flower types**: Daisy, Tulip, 5-Petal Star, Bell Flower, Full Bloom, Wide Daisy, Pointed Star, Mini Cluster
- **3-layer petals**: Dark outer, main color, light inner — with rounded pixel edges
- **24-color palette** with auto-generated dark/light variants per flower
- **Perspective rendering**: Flowers shrink toward the horizon; 3-layer rolling hills for depth
- **Cat animations**: 4-directional sprites (side/front/back), leaping arc, landing squish, tail wag
- **Paw prints**: Squished flowers leave colorful paw prints before respawning
- **Particle effects**: Star bursts, dust, and color fragments on hit
- **8 cloud types** drifting across the sky
- **Score tracking** with cooldown indicator
